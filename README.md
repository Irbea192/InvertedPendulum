# 車輪型倒立振子

## 概要
このプロジェクトは、BMI160 6軸センサとモーター制御IC（TB6612）を用いた倒立振子の制御プログラムです。
## 使用ライブラリ
- [BMI160-Arduino](https://github.com/hanyazou/BMI160-Arduino)  
  IMUセンサ用ライブラリ（LGPL v2.1）
- [MadgwickAHRS](https://github.com/arduino-libraries/MadgwickAHRS)  
  姿勢推定フィルタ
- [SparkFun_TB6612](https://github.com/sparkfun/SparkFun_TB6612FNG_Arduino_Library)  
  モータードライバ用
